# supply-chain-logistics-company

## Industry Type
Supply Chain and Logistics

## Features
- Dark premium industrial UI with bold typography and glowing accents
- Animated hero section, reveal-on-scroll effects, and interactive hover states
- Animated statistics counters and responsive card-based layouts
- Industry-specific interactive tool: Shipping Route Visualizer
- Multi-page structure: Home, About, Services, Projects, Contact

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, transitions, responsive design)
- Vanilla JavaScript (animations and tool logic)

## Portfolio Description
This project is a premium agency-style portfolio website for supply chain and logistics services. It highlights capabilities, representative project outcomes, and an interactive simulator to demonstrate domain-specific value in a visually intense industrial interface.
